## Usage

1. Clone project code.

```bash
git clone https://github.com/keeperdog/antd-admin.git
cd my-project
```

2. Installation dependence.

```bash
yarn install
```

Or

```bash
npm install
```

3. Start local server.

```bash
npm run start
```

4. open a browser and visit [http://localhost:7000](http://localhost:7000)
